Aplicação que ultiliza o Node com sequelize para se conectar ao MYSQL.

tem um controllador pra fazer o CRUD básico 'api/controller'
Rotas em 'api/routes'.

O front-end é usado o 'handlebars' como template engine.

Para executar o programa digitamos na linha do comando 'npm run dev'.

Dependencias:

    "express": "^4.18.2",
    "express-handlebars": "^7.0.7",
    "mysql2": "^3.3.2",
    "path": "^0.12.7",
    "sequelize": "^6.31.1",
    "sequelize-cli": "^6.6.0"
